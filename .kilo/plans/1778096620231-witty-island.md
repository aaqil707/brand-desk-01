# Plan: Fix Profile Live Updates and Remove Video/Excel Logic

## Goal
1. Enable live updates for profiles linked to Google Sheets.
2. Remove video-related sections and inputs.
3. Remove Excel sheet upload and processing logic.

## Step-by-Step Implementation

### 1. Fix Live Profile Updates
The profiles are currently served as static JSON files. To make them "live", `get_profile.php` needs to sync with the Google Sheet if an `empId` is present in the profile.

- **Modify `pages/api/get_profile.php`**:
    - After reading the JSON file, check if `empId` exists in the data.
    - If `empId` exists:
        - Fetch data from the Google Visualization API: `https://docs.google.com/spreadsheets/d/1tOHMOzioUGjjwmJvSlEFHrPrP1hXphk46q8Q_VrejVk/gviz/tq?tqx=out:json&headers=1`.
        - Parse the response (remove the `google.visualization.Query.setResponse(` wrapper).
        - Find the row where the first column matches `empId`.
        - Update the profile data:
            - Name, Designation, Email, LinkedIn, Team Lead, Lead Name.
            - Parse the reviews and ratings (handling the `/n` or `/e` separators as done in the frontend).
        - Save the updated data back to the JSON file.
    - Return the (potentially updated) profile data.

### 2. Remove Video Section
- **`frontend/src/pages/ProfilePage/ProfilePage.jsx`**:
    - Remove the `renderVideo` helper function.
    - Remove the logic that renders the video section based on `profile.videoUrl`.
- **`frontend/src/pages/SignatureGenerator/SignatureGenerator.jsx`**:
    - Remove `videoUrl` state and its `useState` declaration.
    - Remove `videoUrl` from `loadProfileData` logic.
    - Remove `videoUrl` from the `profileData` object in `handleGenerate`.
    - Remove the "Intro Video URL" input field from the JSX.
- **`frontend/src/pages/ProfilePage/ProfilePage.css`**:
    - Remove all CSS rules related to `.video-container`.

### 3. Remove Excel Sheet Logic
- **`frontend/src/pages/SignatureGenerator/SignatureGenerator.jsx`**:
    - Remove the `xlsx` library import: `import * as XLSX from 'xlsx';`.
    - Remove `reviewsFile` state and its `useState` declaration.
    - In `handleGenerate`:
        - Remove the entire block that reads and processes the `reviewsFile` buffer using `XLSX`.
        - Keep the logic that handles `fetchedReview` from Google Sheets.
    - Remove the "Candidate Reviews (Excel File)" input field from the JSX.
